MorphCore Docs

Documentación completa para la librería de modales morph

🚀 Inicio Rápido

1. CDN Requerido

Incluye GSAP en tu proyecto para las animaciones:

<script src=""https://cdn.jsdelivr.net/npm/gsap@3.15/dist/gsap.min.js"></script>

2. Estructura HTML Básica

¿Qué es modal-trigger? Es el elemento que al hacer clic abre el modal. Puede ser cualquier elemento HTML (div, button, etc.) y define el punto de partida de la animación morph. Lo que pongas aquí se "transforma" visualmente en el modal. 
<!-- Fondo oscuro del modal (backdrop) -->
<div id="mo"></div>

<!-- Trigger del modal -->
<div class="modal-trigger" data-w="400" data-h="300">
  <span>Abrir Modal</span>

  <!-- Contenido oculto del modal -->
  <div class="modal-content" style="display:none;">
    <button data-close>×</button>
    <h4>Título del Modal</h4>
    <p>Contenido aquí...</p>
    <button data-close>Cerrar</button>
  </div>
</div>

3. JavaScript Inicialización

<script src=""main.js"></script>
<script>
  // Inicializar MorphCore con GSAP
  MorphCore.init({ gsap });
</script>

🎨 Estilos y Personalización

MorphCore es altamente personalizable. Puedes crear diferentes estilos para diferentes triggers usando clases CSS.

CSS Obligatorio

Estos estilos son esenciales para que funcione correctamente:

/* Estilo básico del trigger */
.modal-trigger {
  cursor: pointer;
  /* Aquí defines cómo se ve el botón/element que abre el modal */
}

/* Contenido del modal cuando está abierto */
.modal-content.is-open {
  padding: 24px;
  display: flex !important;
  flex-direction: column;
  height: 100%;
  box-sizing: border-box;
  /* Estilos del interior del modal */
}

/* Clon del morph (no modificar) */
.morph-clone {
  position: fixed;
  z-index: 1000;
  overflow: hidden;
}

Personalizando modal-trigger

El modal-trigger es el corazón de la interacción. Puedes tener múltiples triggers con estilos diferentes usando clases adicionales.

Ejemplo: Múltiples Triggers con Estilos Distintos

<!-- Trigger estilo botón moderno -->
<div class="modal-trigger btn-primary" data-w="400" data-h="300">
  <span>Configuración</span>
  <div class="modal-content" style="display:none;">...</div>
</div>

<!-- Trigger estilo tarjeta -->
<div class="modal-trigger card-style" data-w="500" data-h="400">
  <span>Perfil</span>
  <div class=class="modal-content" style="display:none;">...</div>
</div>
/* Estilos para diferentes triggers */
.btn-primary {
  background: #007acc;
  color: white;
  padding: 15px 30px;
  border-radius: 25px;
  border: none;
  transition: all 0.3s ease;
}

.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 10px 20px rgba(0,122,204,0.3);
}

.card-style {
  background: #252526;
  border: 1px solid #3e3e42;
  padding: 20px;
  border-radius: 8px;
  color: #d4d4d4;
  min-height: 100px;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: border-color 0.3s ease;
}

.card-style:hover {
  border-color: #569cd6;
}
💡 Tip: Usa clases específicas para cada tipo de modal-trigger. Esto te permite tener diferentes estilos visuales que se adapten al contexto de tu aplicación.

📋 Data Attributes

Atributo Descripción Valores Default Ejemplo
data-w Ancho del modal en píxeles Número positivo 400 data-w="500"
data-h Alto del modal en píxeles Número positivo 400 data-h="350"
data-pos Posición del modal center, origin, custom center data-pos="origin"
data-x Posición X en porcentaje (solo con custom) 0-100 50 data-x="25"
data-y Posición Y en porcentaje (solo con custom) 0-100 50 data-y="75"
data-radius Radio del borde del modal Valor CSS "" data-radius="20px"
data-dur Duración de la animación en segundos Número decimal 0.9 (abrir), 0.7 (cerrar) data-dur="1.2"
data-close Elemento que cierra el modal al hacer clic Atributo presente - data-close

🔧 Consejos y Ejemplos

  • Responsive: Usa media queries para ajustar data-w y data-h en diferentes pantallas.
  • Accesibilidad: Asegúrate de que los botones tengan texto alternativo y usa ARIA labels si es necesario.
  • Performance: MorphCore usa GSAP para animaciones suaves. Evita animar propiedades costosas como box-shadow.
  • Múltiples Modales: Cada modal-trigger puede tener su propio contenido y estilos. Usa IDs únicos si necesitas controlarlos individualmente.
