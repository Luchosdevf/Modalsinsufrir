# MorphModaL

Un mini sistema para crear modales que se expanden de forma bonita y fluida desde cualquier elemento.

Animaciones suaves, stagger y blur en el contenido, guarda el estado de los formularios y se ve bien  sin complicarte la vida.

Solo GSAP + este script y listo.

Hecho para que quede lindo rápido.
